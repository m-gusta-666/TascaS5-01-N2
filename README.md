Requests vàlides: 

GET http://localhost:9001/swagger/flor/getAll

POST http://localhost:9001/swagger/flor/add
PUT http://localhost:9001/swagger/flor/update/{id}
DELETE http://localhost:9001/swagger/flor/delete/{id}
